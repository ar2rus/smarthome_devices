Инструкция по сборке проектов в Atmel Studio c подключаемыми библиотеками на примере:

1.В проекте создать папку clunet и в нем создать только clunet_config.h с необходимыми параметрами.

2.Добавляем в директории проекта папку _lib целиком и только что созданную папку clunet
Project properties->AVR/GNU C Compilier Directories.>+

Не забыть поставить Relative path

Пути в Include Paths (-I) после выполнения этого пункта выглядят так
../../../_lib
../clunet

3.Добавляем в проекте в папку clunet все необходимые файлы этой библиотеки, но КАК ССЫЛКУ

правый клик по папке clunet в дереве проектов Add->Existing Item выбрать файлы clunet.c и clunet.h из ../../../_lib/clunet и нажать Add as Link (выпадающее меню из кнопки Add)


Таким образом можно файл конфига, находящийся в проекте, для общей для нескольких проектов библиотеки.
